해야 할 일:

```text
날짜 Quick Action Overlay에서 할 일 완료 toggle 연결
Quick Capture 기본 저장 모드를 task로 둘 것
날짜 지정/시간 지정 기존 정책을 유지할 것
TaskPanel legacy 사용 여부를 검색 후 정리할 것
```

주의:

```text
deletedAt !== null row는 UI에 표시하지 않는다.
```
